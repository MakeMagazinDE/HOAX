HOAX-Tuning
===========

LabVIEW-VIs zur bequemen Einstellung aller HX3-Paramater.

Diese Programme werden ohne Gewähr zur Verfügung gestellt; die Anwendung geschieht auf eigene Gefahr.
Sie benötigen ein installiertes LabVIEW 6.1 zur Ausführung (Vollversion auf Heft-DVD c't 11/2007).
Erfahrung mit LabVIEW ist ggf. erforderlich.
